USER STORY:- As a sportsperson, I want to know about the location and quality of water of a nearby water resource when I travel to new locations, so that I can access water easily whenever I need it rather than searching for a shop.

HOME SCREEN:- introduces the starting page to the user when the app is opened.It contains the list of features and basic options like settings etc for the users to choose from.

SEARCH SCREEN:- after choosing the water locator feature of the app, this window appears .The user enters the location and then the nearby water resources are shown on the map along with their listed view.

DIRECTION SCREEN:- this screen appears when the user clicks on the direction tab, after choosing one of the water source from the search screen.This screen helps the user to trace the path to the desired location
